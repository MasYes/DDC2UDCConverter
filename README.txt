This program created for converting DDC (Dewie Decimal Classification) to UDC (Universal Decimal Classification).

All you need is to download this library and to import this into your project.

Also, we want to add all of the DDC numbers and to add other classifications.

Example of converting DDC to UDC.

Converter conv = new Converter();
conv.DDC2UDC("125");


Example of converting UDC to DDC.

Converter conv = new Converter();
conv.UDC2DDC("27-312.55");
